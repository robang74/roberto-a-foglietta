How to install Google Play on FireOs 7 (e.g. on Fire 10 HD, 2021)

Created on 2022-11-08 13:36

Published on 2022-11-08 13:43

STEP-BY-STEP INSTALLATION

A) Allows the installation of application from unknown source

  1. Open Device Options

  2. Security & Privacy

  3. Apps from unknown ource

  4. Silk Browser (not allowed)

  5. Allow Silk Browser

B) Download and install these four components for Andorid 9+

  1. Google Account Manager

  2. Google Service Framework

  3. Google Play Service

  4. Google Play Store

C) Revert allowance at point A, reboot and sign-in into the Google Play Store

D) Install Google applications and whatever you like

UPDATE 15.11.2022

Do not use Google Chrome browser because it clogs the system but continue using Amazon Silk as default browser. A good alternative to Silk is DuckDuckGo the open-source privacy oriented browser.

Enjoy, R-
